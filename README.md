Безопасная панель управления для множества VPN-серверов.

## ✨ Возможности

- 🖥 Управление несколькими серверами из одной панели
- 👥 Создание и управление клиентами VPN
- 🔒 Безопасная авторизация (Argon2id)
- 📱 QR-коды для мобильных устройств
- 🛡 Защита от атак (SQL-инъекции, XSS, CSRF)
